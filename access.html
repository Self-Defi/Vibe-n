Wallet Connect

After you pay, you’ll verify purchase, then unlock access.

  1. Complete Fiat or Crypto payment.
  2. Save proof: receipt (fiat) or tx hash (crypto).
  3. Connect wallet (your gating button / portal).
  4. Submit proof (receipt ID / tx hash) → access is granted.
  5. If crypto: confirm the network matches the invoice network before sending.
