Buy crypto → send payment

Simple flow. Don’t rush network selection.

Steps
  1. Buy USDC (or required asset) on your preferred on-ramp.
  2. Confirm the target network (Base / Ethereum / Polygon) matches the invoice.
  3. Send → paste invoice address → set amount → confirm.
  4. Save tx hash, then go to Wallet Connect to unlock.

Rule: network must match the invoice network. If you mess up, funds can be stuck.
