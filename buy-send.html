Rule: network must match the invoice network

If you send on the wrong network, funds can be stuck. Slow down here.

Option A — Coinbase (beginner friendly)

  • Buy USDC (or the required asset).
  • Tap Send → paste the payment address.
  • Select the correct network (Base / Ethereum / Polygon).
  • Confirm → save the tx hash for proof.

Option B — MetaMask (EVM)

  • Get ETH for gas + USDC for payment.
  • Open MetaMask → Send.
  • Paste address → set amount → confirm.
  • Save tx hash, then wallet-connect to unlock.
